### Readme for running the sarsa agent and the q learning agent. 

1. Go to the sarsa folder to run the SARSA agent and to the q_learning folder for the q_Learning agent.
2. Go to the main.py file to change the parameters like num_episodes, Num_agents, Driver threshold, discount factor. 
3. Run main.py using "python main.py"
4. Results Expected: A plot of the reward obtained by the agent in eache episode averaged over the number of trials. 
5. Thank you.
